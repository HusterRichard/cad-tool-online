# 演示材料

本目录包含 CadToolOnline 项目的演示材料，用于展示项目核心功能和使用场景。

## 目录说明

### demo.mp4
**[待录制]** 项目演示视频

**内容要求**:
- 时长: 3-5分钟
- 分辨率: 1920x1080
- 格式: MP4

**演示流程**:
1. **启动插件** (0:00-0:30)
   - 在 VSCode 中打开插件
   - 展示主界面布局

2. **导入 STEP 文件** (0:30-1:30)
   - 打开文件对话框
   - 选择示例 STEP 文件
   - 展示导入进度
   - 显示 3D 模型渲染结果

3. **创建刚体组** (1:30-2:30)
   - 选择几何体
   - 创建 MbsGroup
   - 查看质量属性（质心、惯性矩阵）

4. **定义运动副** (2:30-3:30)
   - 创建坐标系 (Frame)
   - 定义转动副 (Revolute Joint)
   - 展示约束关系

5. **导出 Modelica** (3:30-4:00)
   - 导出模型
   - 展示生成的 .mo 文件

6. **其他功能** (4:00-5:00)
   - Ribbon 菜单导航
   - 模型树操作
   - 视图控制（旋转、缩放、适配）

## 使用说明

### 录制演示视频

**推荐工具**:
- Windows: OBS Studio, ShareX
- macOS: QuickTime Player, ScreenFlow
- Linux: SimpleScreenRecorder, Kazam

**录制步骤**:
1. 准备测试数据（STEP 文件）
2. 清理 VSCode 工作区，关闭无关文件
3. 设置屏幕分辨率为 1920x1080
4. 开始录制，按照演示流程操作
5. 录制完成后转换为 MP4 格式
6. 重命名为 `demo.mp4` 并放置在此目录

### 截取屏幕截图

**截图要求**:
- 格式: PNG
- 分辨率: 原始分辨率（建议 ≥1920x1080）
- 命名: 按照上述规范命名

**截图技巧**:
- 使用高对比度主题，便于查看
- 确保截图内容清晰，文字可读
- 包含关键 UI 元素（菜单、工具栏、面板）
- 避免包含敏感信息（文件路径、用户名等）
