# Commit title format:
# <type>: <task id> <summary>
#
# Rules:
# - task id is required
# - if there is no real task id, use T000
# - use Conventional Commit types
#
# Examples:
# feat: T013 新增装配体导入校验
# fix: T102 修复 STEP 层级名称解析
# chore: T000 统一提交消息校验规则
#
# Recommended body:
# - key change 1
# - key change 2
#
# Co-authored-by: <Name>
