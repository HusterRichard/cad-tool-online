# 任务书

工作区中有 3 个项目，现在我们要完成一个将 C++ 工具转换成 VSCode 插件工具的开发工作。

## 1. 输入项目

### 1.1 CADToolbox

是一个由 Qt 5.14.2 开发的桌面应用，几何引擎使用的是 OCCT 7.7.0，可以读三维 CAD 文件（例如 STEP 文件），提供三维 CAD 图形设计环境，支持分组设计、标架设计、关节设计和驱动设计等多体动力学的设计能力，可以自动提取三维 CAD 模型的几何信息和物理属性，最终转换成多体动力学系统模型，可以表达成多刚体之间约束关系图形。

### 1.2 chili3d

是一个 Online 版的三维 CAD，几何引擎使用的是 OCCT，显示使用的是 three.js，通过将 OCCT 和上层 C++ 写的应用接口编译成 wasm 文件后，实现了在线调用 OCCT 内核的功能。

## 2. 输出项目

### 2.1 CadToolOnline

将单机版的 CADToolbox 转换成在线版，形态是一个可以在 VSCode 上面安装使用的插件，支持导入 STEP 格式的三维 CAD 模型，提供支持分组设计、标架设计、关节设计和驱动设计等多体动力学的设计能力，并能转换 MxGraph 显示的多体动力学关系拓扑图形。

**两种可选方案：**

- **方案一**：将 CADToolbox 的 C++ 代码编译成 wasm 文件，注意该工具还依赖了 Sysplorer SDK，这些依赖最好全部都要去掉，否则 wasm 文件会特别大。
- **方案二**：只将 OCCT 编译成 wasm，其它的应用层代码全部用前端 JS 代码重写。
