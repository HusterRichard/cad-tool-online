## 8. 关键测试场景

### 8.1 入口与面板

1. 点击 `转动驱动`
   - 期望: 右侧打开 `选项-驱动`
   - 期望: 默认 `闪电模式`
2. 点击 `平移驱动`
   - 期望: 右侧打开 `选项-驱动`
   - 期望: 类型为 `平移驱动`

### 8.2 闪电模式

1. 进入 `转动驱动`
2. 选择一个转动副
   - 期望: 立即创建驱动
   - 期望: 驱动出现在模型树 `驱动` 分类
3. 再选择另一个转动副
   - 期望: 可连续创建第二个驱动

### 8.3 标准模式

1. 进入驱动创建态
2. 切换到 `标准模式`
3. 选择一个兼容连接
   - 期望: 只更新草稿，不立即创建
4. 点击 `添加`
   - 期望: 创建成功

### 8.4 兼容性校验

1. 在 `旋转驱动` 模式下选择平移副
   - 期望: 禁止创建，给出不兼容提示
2. 在 `平移驱动` 模式下选择转动副
   - 期望: 禁止创建，给出不兼容提示
3. 在两种模式下分别选择圆柱副
   - 期望: 都允许创建

### 8.5 属性面板

1. 选中已创建驱动
   - 期望: 右侧显示 `属性-驱动`
2. 检查字段
   - 期望: 存在 `名称 / 可见性 / 图标大小 / 驱动类型 / phi.start / w.start`
   - 期望: `类型` 为只读
3. 修改驱动类型
   - 旋转驱动: `角度 / 角速度 / 角加速度`
   - 平移驱动: `位移 / 速度 / 加速度`
4. 保存
   - 期望: 模型树名称和属性值同步更新

### 8.6 快捷键与右键菜单

1. 创建态按 `Esc`
   - 期望: 退出添加驱动状态
2. 选中驱动按 `Del`
   - 期望: 驱动删除
3. 模型树驱动节点右键
   - 期望: 可见 `属性 / 删除`

### 8.7 图标大小

1. 在创建态手动输入图标大小
   - 期望: 草稿更新
2. 在创建态拖动图标大小滑条
   - 期望: 草稿更新
3. 在创建态按 `Alt + 鼠标滚轮`
   - 期望: 图标大小增减并刷新面板

### 8.8 级联删除

1. 创建一个依附某连接的驱动
2. 删除该连接
   - 期望: 关联驱动自动删除
   - 期望: 模型树无残留驱动节点

### 8.9 导入导出

1. 导出包含驱动的配置
   - 期望: `motion` 保留 `driveMode / iconSize / visible / phiStart / wStart`
2. 导入旧字段别名
   - 期望: `functionType / visibility / start / w` 能被兼容解析
3. 导入未知连接引用
   - 期望: 给出 warning，不导致整体导入失败

## 10. 已知差异登记

### 10.1 与帮助页目标态差异

- 当前 MVP 未提供独立驱动三维图标对象。
- 当前驱动创建主要依附于“连接选择”，而不是独立驱动操纵器。

### 10.2 可接受前提

- 只要驱动创建、属性编辑、删除、导入导出闭环成立，且帮助页要求的关键行为落地，即可通过 `SC06 MVP` 验收。
