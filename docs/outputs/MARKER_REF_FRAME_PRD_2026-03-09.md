# PRD: 添加标架及参考标架

日期: 2026-03-09
项目: CadToolOnline
来源文档:
- `ref/Docs/CADToolBox/Doc/CADToolBox/HowToUseCADToolBox/BuildMarkers.html`
- `ref/Docs/CADToolBox/Doc/CADToolBox/HowToUseCADToolBox/ObjectAttribute.html`

## 1. 背景

当前在线版已经具备最小可用的标架与参考标架导入/导出能力，但与帮助文档定义的操作流程存在明显差距：
- 标架仅支持“选中零件后点一下即创建”，缺少闪电模式和标准模式完整流程。
- 参考标架错误地依赖三维拾取放置，而帮助文档要求“选择基本标架 + 选择目标零件所属组”即可完成。
- 模型树中没有标架/参考标架节点，无法通过模型树完成选择、属性查看和二次操作。
- 属性侧边栏未覆盖帮助文档要求的可编辑项和只读项。

这会直接影响多体接口设计，尤其是弹簧、外部接口、设计点等后续功能的稳定性。

## 2. 产品目标

实现与帮助文档一致的“添加标架及参考标架”能力，使用户可以在模型树与三维视图之间完成以下闭环：
- 创建基本标架。
- 创建与基本标架关联的参考标架。
- 在模型树中查看和选择标架/参考标架。
- 在属性面板中编辑基本标架。
- 在属性面板中查看参考标架，并仅编辑可见性。
- 在保存/打开配置后保留标架/参考标架关键属性。

## 3. 术语

- 基本标架: 独立创建在零件或分组上的标架。
- 参考标架: 关联某个基本标架、附着于另一目标零件所属组的标架。
- 闪电模式: 满足必要输入后，拾取或选择目标即立即创建。
- 标准模式: 满足必要输入后，需要显式点击“添加”按钮才创建。
- 目标零件所属组: 选中的目标零件归属的分组；若零件未分组，则视该零件自身为目标宿主。

## 4. 用户故事

### 4.1 基本标架

作为多体设计用户，我希望在完成零件分组后，可以通过菜单进入“标架”状态，在选定零件表面拾取位置与方向，从而快速创建一个可作为外部接口的基本标架。

### 4.2 参考标架

作为多体设计用户，我希望在已有基本标架的前提下，通过选择基本标架和另一个目标零件，快速生成一个与该基本标架关联的参考标架，用于表达同一接口在不同组之间的从属关系。

### 4.3 属性编辑

作为多体设计用户，我希望在模型树中选中标架或参考标架后，右侧属性面板能显示对应属性，并允许我按帮助文档约束进行编辑。

- Ribbon 入口:
  - `标架`
  - `参考标架`

  - 闪电模式
  - 标准模式
  - 名称
  - 图标大小
  - 位置
  - 方向
  - 可见性
- 参考标架:
  - 闪电模式
  - 标准模式
  - 基本标架选择
  - 目标零件所属组选择
  - 同顶层组限制
  - 可见性
- 模型树:
  - 标架节点
  - 参考标架节点
  - 选中联动
- 属性面板:
  - 基本标架可编辑项
  - 参考标架只读项和可见性编辑
- 配置持久化:
  - 名称

### 5.2 本次不做

- 点/线/面的高阶特征智能推理算法增强。
- 基于 Euler 角的单独姿态编辑器。
- 参考标架批量创建。
- 参考标架独立改名、改位置、改方向。
- 设计点、连接、驱动对新属性字段的额外联动改造。

## 6. 交互需求

### 6.1 创建基本标架

#### 闪电模式

1. 用户点击 Ribbon `标架`。
2. 右侧弹出 `选项-标架` 面板。
3. 用户可编辑名称、图标大小、模式开关。
4. 用户在三维视图中对当前选中零件执行拾取。
5. 系统根据拾取结果生成位置和 Z 方向。
6. 系统立即创建基本标架。
7. 用户可连续重复第 4 步，继续创建。
8. 用户再次点击 Ribbon `标架` 或按 `Esc` 退出。

#### 标准模式

1. 用户点击 Ribbon `标架` 并切换到 `标准模式`。
2. 用户在三维视图中拾取放置点，系统仅更新草稿标架。
3. 用户可通过“拾取位置”或“拾取方向”再次更新草稿。
4. 用户点击 `添加` 后，系统正式创建基本标架。
5. 用户可继续下一轮创建或取消退出。

### 6.2 创建参考标架

#### 闪电模式

1. 用户点击 Ribbon `参考标架`。
2. 右侧弹出 `选项-参考标架` 面板。
3. 用户选择一个基本标架，可来自模型树或三维视图。
4. 用户选择一个目标零件，可来自模型树或三维视图。
5. 系统取该目标零件所属组作为参考标架宿主。
6. 若目标宿主与基本标架位于同一顶层组，则禁止创建并提示。
7. 否则系统立即创建参考标架，位置和方向默认复制基本标架。

#### 标准模式

1. 用户切换到 `标准模式`。
2. 用户选择基本标架。
3. 用户选择目标零件。
4. 用户点击 `添加`。
5. 系统执行与闪电模式相同的校验和创建。

### 6.3 模型树

- 标架与参考标架作为对象节点显示在其宿主分组或零件下。
- 选中标架节点时，三维视图同步高亮对应标架。
- 选中参考标架节点时，三维视图同步高亮对应参考标架。
- 当通过三维视图选中标架/参考标架时，模型树同步选中对应节点。

### 6.4 属性面板

#### 基本标架

可编辑项:
- 名称
- 可见性
- 图标大小
- 位置
- 方向

交互要求:
- 数值修改后需显式保存。
- 支持通过“拾取位置”“拾取方向”更新对应字段。
- 回车更新坐标或方向时立即刷新界面显示。

#### 参考标架

只读项:
- 名称
- 基本标架
- 宿主对象
- 位置
- 方向

可编辑项:
- 可见性

## 7. 数据规则

### 7.1 宿主归属

- 若零件已分组，基本标架归属该分组。
- 若零件未分组，基本标架归属该零件自身。
- 参考标架归属目标零件所属组；若目标零件未分组，则归属该零件自身。

### 7.2 参考标架约束

- 创建参考标架前必须已存在至少一个基本标架。
- 参考标架必须关联一个基本标架。
- 参考标架不得附着到与基本标架相同的顶层组。
- 删除基本标架前，若仍存在关联参考标架，则必须阻止删除。

### 7.3 持久化

导出配置时，标架与参考标架需保留：
- `name`
- `groupRef`
- `position`
- `direction`
- `size`
- `visible`
- `refMarker`
- `relatedMarkerRef`

导入配置时，缺失字段使用兼容默认值：
- `size`: `-1`
- `visible`: `true`
- `direction`: `+Z`
- `position`: `(0,0,0)` 或关联基本标架的位置

## 8. 验收标准

### 8.1 基本标架

- 能从 Ribbon `标架` 进入创建状态。
- 闪电模式下拾取一次即可创建。
- 标准模式下必须点击 `添加` 才会正式创建。
- 标架会出现在模型树正确宿主节点下。
- 选中标架后属性面板可编辑名称、可见性、图标大小、位置、方向。

### 8.2 参考标架

- 能从 Ribbon `参考标架` 进入创建状态。
- 闪电模式下完成“基本标架 + 目标零件”后立即创建。
- 标准模式下必须点击 `添加` 才会正式创建。
- 参考标架位置与方向默认复制基本标架。
- 同顶层组场景被拦截并提示。
- 参考标架出现在模型树正确宿主节点下。
- 选中参考标架后只能改可见性，其他字段只读。

### 8.3 联动与持久化

- 模型树与三维视图可双向同步选中标架/参考标架。
- 保存后重新打开配置，标架/参考标架名称、宿主、可见性、大小和关联关系保持一致。

## 9. 风险与约束

- 当前拾取能力依赖已选中零件和面法向计算，无法完全复刻桌面版的自由悬停特征识别。
- 参考标架在本次实现中采用“复制基本标架初始位姿”的规则，以匹配帮助文档“仅选择目标零件即可添加”的主流程。
- 三维视图对标架对象的选择需要扩展当前 `SelectionManager` 的注册对象范围。
