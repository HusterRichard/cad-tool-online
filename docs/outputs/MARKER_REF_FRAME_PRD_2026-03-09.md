# PRD: 添加标架及参考标架
日期: 2026-03-09
项目: CadToolOnline
更新: 2026-03-10

来源文档:

## 1. 背景

当前仓库已经具备标架与参考标架的基础数据结构、三维显示、导入导出和部分属性编辑能力，但与帮助文档中的设计流程仍存在偏差，主要体现在：
- 点击 Ribbon `标架` 后，不应要求预先选中零件，而应立即进入设计状态并显示 `选项-标架`。
- 鼠标在零件上拾取时，三维视图需要同步显示标架预览，而不是等点击后才出现。
- 智能拾取需要覆盖线段端点、柱面轴线点、球心三类高频几何特征。
- 创建完成后，模型树需要在宿主零件节点下生成标架节点，并支持树选中与三维对象联动。
- `属性-标架` 面板需要支持完整编辑名称、可见性、大小、位置和方向。

这些交互差异会直接影响后续连接、副、设计点等多体设计功能的可用性。

## 2. 产品目标

实现与帮助文档一致的标架设计主流程，使用户可以在 Ribbon、三维视图、模型树和属性面板之间完成完整闭环：
- 点击 Ribbon `标架` 后立即进入设计状态
- 鼠标悬停时实时显示标架预览
- 点击零件表面后创建标架并挂接到对应零件节点
- 通过模型树选择、右键删除和三维选择完成标架管理
- 通过 `属性-标架` 面板编辑名称、可见性、大小、位置和方向
- 在已有基本标架基础上创建参考标架

## 3. 术语

- 基本标架: 用户直接创建在零件上的标架。
- 参考标架: 关联某个基本标架，并附着在目标零件上的标架。
- 宿主组: 标架在多体设计语义上的归属组；若零件未分组，则退化为零件自身。
- 宿主零件: 标架在模型树中的直接挂载零件节点。
- 智能推断: 根据几何特征自动修正标架位置与方向。

## 4. 用户故事

### 4.1 基本标架

作为多体设计用户，我希望点击 Ribbon `标架` 后立即看到 `选项-标架`，并在鼠标悬停零件时看到标架预览，这样我可以直观确认创建位置和方向。

### 4.2 标架管理

作为多体设计用户，我希望创建后的标架显示在宿主零件节点下，并支持模型树选中同步三维标架对象、右键删除标架。

### 4.3 属性编辑

作为多体设计用户，我希望在 `属性-标架` 中修改名称、可见性、大小、位置和方向，并立即同步刷新模型树与三维显示。

### 4.4 参考标架

作为多体设计用户，我希望在已有基本标架后，通过选择基本标架和目标零件快速创建参考标架，用于描述跨零件、跨组的对应关系。

## 6. 交互需求

### 6.1 基本标架主流程

1. 用户点击 Ribbon `标架`。
2. 右侧立即显示 `选项-标架`。
3. 系统进入标架设计模式后，三维视图不再响应零件选中拾取，改为标架拾取与放置交互。
4. 鼠标拾取零件时，零件表面同步显示标架预览。
5. 在开启 `智能推断` 时，系统自动识别：
   - 线段起始点和终点
   - 柱面的轴线点
   - 球面的中心
6. 用户点击鼠标后，在推断结果位置正式创建标架。
7. 左侧模型树在对应零件节点下新增标架节点。
8. 只要用户没有主动退出标架设计模式，系统继续保持标架拾取与放置交互，不恢复零件选中响应。

### 6.2 基本标架标准模式

1. 用户切换到 `标准模式`。
2. 鼠标悬停时显示预览。
3. 点击后更新草稿标架。
4. 用户可通过 `拾取放置`、`拾取位置`、`拾取方向` 继续修正。
5. 点击 `添加` 后正式创建标架。

### 6.3 参考标架流程

1. 用户点击 Ribbon `参考标架`。
2. 右侧立即显示 `选项-参考标架`，并进入参考标架设计状态。
3. 用户在三维视图中拾取一个基本标架对象，系统将其设为当前基本标架。
   - 右侧 `选项-参考标架` 面板需立即刷新并显示该源标架名称。
4. 用户可在 `闪电模式` 与 `标准模式` 之间切换，切换模式不清空已选基本标架。
5. 用户在模型树或三维视图中选择目标零件，且目标零件所属顶层组不能与基本标架宿主组相同：
   - 右侧 `选项-参考标架` 面板需立即刷新并显示目标零件名称。
   - `闪电模式`: 选中目标零件后立即尝试创建参考标架。
   - `标准模式`: 选中目标零件后，点击 `添加` 才正式创建。
6. 系统在创建前执行跨组校验；校验失败时阻止创建并提示原因。
7. 创建成功后，在目标零件节点下生成参考标架节点，并在三维视图中高亮该对象。
8. 参考标架图标必须与基本标架图标不同，且参考标架名称与基本标架名称一致。
9. 只要用户没有主动取消或切换到其他模式，系统保持参考标架设计状态，支持连续创建。

### 6.4 模型树联动

- 标架和参考标架节点显示在宿主零件节点下。
- 选中模型树标架节点时，三维视图同步选中对应标架对象。
- 通过三维视图选中标架对象时，模型树同步高亮对应节点。
- 标架节点和参考标架节点右键菜单必须提供 `删除`。

### 6.5 属性面板

#### 属性-标架

可编辑项:
- 名称
- 可见性
- 标架大小
- 位置
- 方向

交互要求:
- 保存后同步刷新三维标架对象与模型树名称。
- 支持 `拾取位置`、`拾取方向` 二次修正。

#### 属性-参考标架

只读项:
- 名称
- 基本标架
- 宿主对象
- 位置
- 方向

可编辑项:
- 可见性

## 7. 数据规则

### 7.1 归属规则

- `groupId` 表示标架的多体语义宿主组。
- `parentId` 表示标架在模型树中的直接宿主零件。
- 基本标架创建时必须同时记录 `groupId` 和 `parentId`。
- 参考标架创建时必须同时记录目标宿主组和目标宿主零件。

### 7.2 智能推断规则

- 默认开启 `智能推断`。
- 线段推断:
  - 优先吸附到最近端点
  - 方向沿线段方向
- 柱面推断:
  - 位置投影到柱面轴线
  - 方向沿柱面轴线
- 球面推断:
  - 位置吸附到球心
  - 方向沿球心到命中点的径向
- 推断失败时，回退到命中点 + 面法向。

### 7.3 持久化规则

导出配置时保留:
- `name`
- `groupRef`
- `partRef`
- `position`
- `direction`
- `size`
- `visible`
- `relatedMarkerRef`

导入配置时:
- `partRef` 优先恢复模型树宿主零件
- `groupRef` 恢复语义宿主组
- 缺失 `partRef` 时，可退回到 `groupRef`

## 8. 验收标准

### 8.1 基本标架

- 打开模型后默认不自动选中全部零件。
- 点击 Ribbon `标架` 后立即出现 `选项-标架`。
- 启动标架设计模式后，三维视图不再响应零件选中，只响应标架拾取与放置。
- 连续创建多个标架期间，系统持续保持标架设计模式，直到用户显式取消或切换模式。
- 鼠标悬停零件时出现标架预览。
- 点击线段附近时，标架吸附到最近起点或终点。
- 点击柱面时，标架吸附到轴线点且方向沿轴线。
- 点击球面时，标架吸附到球心且方向沿径向。
- 点击创建后，模型树在对应零件下新增标架节点。

### 8.2 模型树与属性

- 选中标架节点时，三维视图同步选中对应标架对象。
- 标架节点右键菜单可以删除标架。
- `属性-标架` 支持修改名称、可见性、大小、位置、方向。
- 保存属性后，模型树和三维视图同步刷新。

### 8.3 参考标架

- 点击 Ribbon `参考标架` 后立即出现 `选项-参考标架`。
- 进入参考标架设计状态后，必须先在三维视图中拾取一个基本标架对象。
- 选中基本标架后，`选项-参考标架` 面板实时显示源标架名称。
- `闪电模式` 下，目标零件选定后立即创建参考标架。
- `标准模式` 下，目标零件选定后点击 `添加` 才创建参考标架。
- 选中目标零件后，`选项-参考标架` 面板实时显示目标零件名称。
- 同顶层组场景会被阻止并提示明确原因。
- 创建成功后，在目标零件节点下生成参考标架节点，且三维视图与模型树同步选中。
- 参考标架图标与基本标架图标不同，且参考标架名称与基本标架名称一致。
- 在未取消前可连续创建多个参考标架，不自动退出参考标架设计状态。
- `属性-参考标架` 只允许编辑可见性。

## 9. 风险与约束

- 线段端点吸附当前基于已有 edge 数据近似实现，精度依赖边离散质量。
- 柱面/球面识别依赖 OCCT 几何查询能力与 WASM 绑定。
- hover 预览会提升几何拾取频率，需要关注大模型下的交互性能。

## 10. 2026-03-11 补充交互需求

### 10.1 柱面 hover

- 当鼠标 hover 到柱面时，系统应识别为柱面特征，并绘制柱面的辅助轮廓线。
- 标架放置点应吸附到柱面轴线上的对应中心位置，不再停留在柱面表面命中点。
- 标架中心圆法向必须与柱面轴线一致，因此标架中心圆平面应垂直于柱面轴线。

### 10.2 圆线 hover

- 当鼠标 hover 到圆线时，系统应优先识别圆线特征，并绘制圆线辅助线。
- 标架放置点应吸附到圆心位置。
- 标架中心圆必须与圆线共圆心，并沿圆线法向建立方向。
