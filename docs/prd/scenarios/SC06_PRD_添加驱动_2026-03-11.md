## 3. 背景与目标

驱动是多体设计里把“连接”转成“可运动约束”的关键元素。帮助页显示，用户在完成转动副、平移副、圆柱副等连接后，需要继续添加驱动，才能形成完整的运动链。

`SC06` 的目标不是单纯新增一个菜单按钮，而是补齐以下工作流闭环：

- Ribbon 入口 `转动驱动 / 平移驱动`
- 右侧 `选项-驱动`
- `闪电模式 / 标准模式`
- 基于连接选择创建驱动
- `属性-驱动`
- 模型树中的驱动节点选择、删除
- `Esc` 退出与 `Del` 删除
- `cadtool.config` 导入导出

## 4. 帮助页目标态梳理

### 4.1 BuildMotions

帮助页明确了以下行为：

- 驱动在添加转动副、平移副、圆柱副之后创建。
- 支持 `闪电模式` 和 `标准模式`，默认是 `闪电模式`。
- 创建前右侧打开 `选项-驱动`。
- 用户可配置 `名称`、`驱动类型`、`图标大小`。
- `图标大小` 支持数值输入、滑条、`Alt + 鼠标滚轮`。
- 闪电模式下，选择符合条件的连接后立即创建。
- 标准模式下，选择连接后需要点击 `添加`。
- 再次点击命令或按 `Esc` 退出添加驱动状态。

### 4.2 ObjectAttribute

帮助页明确驱动属性页至少包含：

- `名称`
- `可见性`
- `图标大小`
- `驱动类型`
- `phi.start`
- `w.start`

同时说明：

- 驱动本身的 `类型` 为只读。
- 旋转驱动支持 `角度 / 角速度 / 角加速度`。
- 平移驱动支持 `位移 / 速度 / 加速度`。

### 4.3 ViewInteractions

帮助页明确：

- `Esc` 可退出添加驱动。
- `Del` 可删除当前选中的设计元素。
- 驱动在三维视图和模型树中都属于可选设计元素。

### 4.4 应用案例证据

- `Bulldozer.html`
  - 在两个平移副上添加两个平移驱动。
  - 创建后还要进入属性面板把驱动类型改成 `位移`。
- `PickAndPlace.html`
  - 点击 `转动驱动` 后，右侧打开 `选项-驱动`。
  - 状态提示聚焦到“转动副/圆柱副”，通过选择连接完成创建。
- `Robot.html`
  - 需要连续添加多个转动驱动。
  - 创建后统一把驱动类型改成 `角度`。

## 5. 用户与典型场景

### 5.1 目标用户

- 完成多体连接设计后，需要为机构施加驱动的建模工程师
- 需要把 CAD 机构转为可仿真的 Modelica 多体模型的系统工程师

### 5.2 典型场景

- 为转动副添加旋转驱动
- 为平移副添加平移驱动
- 为圆柱副添加旋转驱动或平移驱动
- 在连续工作流中批量添加多个驱动
- 创建后通过属性面板调整驱动类型、图标大小和初值

## 6. 范围定义

### 6.1 In Scope

- Ribbon `转动驱动`、`平移驱动`
- `选项-驱动`
- `闪电模式 / 标准模式`
- 基于连接选择的驱动创建
- 驱动与连接类型兼容性校验
- 模型树驱动节点
- `属性-驱动`
- `Del` 删除驱动
- `Esc`、关闭面板、再次点击 Ribbon 退出
- `cadtool.config` 中 `motion` 导入导出

### 6.2 Out of Scope

- 帮助页中的驱动图标三维外观完全复刻
- 由驱动直接生成 Modelica 源码的完整下游行为
- 全局设计元素统一图标配置对话框
- 三维视图中专门的驱动独立图标拾取器

## 7. 当前版本 MVP 定义

本次 `SC06 MVP` 采用“连接即驱动来源”的实现方式：

- 右侧提供完整 `选项-驱动` 工作流。
- 通过三维视图或模型树选择连接来完成驱动创建。
- 只允许驱动兼容连接：
  - 旋转驱动: `转动副 / 圆柱副`
  - 平移驱动: `平移副 / 圆柱副`
- 创建后驱动进入模型树 `驱动` 分类，并支持属性编辑、删除、导入导出。

与帮助页目标态相比，当前 MVP 的主要差异是：

- 已支持与关联连接锚定的驱动草稿预览和已创建驱动绘制图形，但仍未复刻帮助页中的专用驱动操纵器。
- 驱动选择依赖已有连接对象，而不是单独的驱动拾取器。

## 8. 功能需求

### 8.1 入口

- 用户点击 Ribbon `转动驱动` 时，进入旋转驱动创建态。
- 用户点击 Ribbon `平移驱动` 时，进入平移驱动创建态。
- 再次点击相同入口可退出当前驱动创建态。

### 8.2 选项面板

`选项-驱动` 至少包含：

- 名称
- 类型只读展示
- 可见性
- 图标大小
- 图标大小滑条
- 连接显示区
- 驱动类型
- `phi.start`
- `w.start`
- `闪电模式 / 标准模式`
- `重置 / 取消 / 添加`

### 8.3 连接选择与创建

- 闪电模式默认开启。
- 选择连接后：
  - 闪电模式立即创建。
  - 标准模式只更新草稿，点击 `添加` 后创建。
- 选择兼容连接时，三维视图需要显示与连接锚定的驱动草稿预览。
- 驱动创建生效后，三维视图中的已创建驱动需要显示为红色。
- 驱动预览和已创建驱动都必须使用代码绘制图形，而不是复用现有 `motion_cad_*.svg` 图标。
- `转动驱动` 需要显示带双线箭头头部的弧形箭头轮廓，且轮廓顶部保持朝上、箭头头部稳定位于右下象限，视觉与帮助页参考图一致。
- 当驱动类型为 `平移驱动` 且选择 `平移副 / 圆柱副` 时，驱动图形中的单向箭头需要与连接轴线平行并共轴。
- 驱动创建态下点击连接不应弹出 `No face found at click position...` 这类面拾取错误提示。
- 连接兼容性不满足时，禁止创建并给出提示。

### 8.4 属性面板

`属性-驱动` 至少支持：

- 修改名称
- 修改可见性
- 修改图标大小
- 修改驱动类型
- 修改 `phi.start`
- 修改 `w.start`
- 删除驱动

### 8.5 模型树与快捷键

- 创建成功后，驱动节点出现在 `驱动` 分类下。
- 选中驱动节点时，右侧显示 `属性-驱动`。
- 驱动节点支持右键 `属性 / 删除`。
- `Del` 删除当前选中驱动。
- `Esc` 退出驱动创建态。

### 8.6 导入导出

`motion` 数据至少保留：

- `name`
- `motionType`
- `connectorRef`
- `driveMode`
- `iconSize`
- `visible`
- `phiStart`
- `wStart`

## 9. 验收标准

1. 点击 `转动驱动` 或 `平移驱动` 后，右侧出现 `选项-驱动`。
2. 默认进入 `闪电模式`。
3. 旋转驱动只能绑定转动副或圆柱副。
4. 平移驱动只能绑定平移副或圆柱副。
5. 选择兼容连接时，三维视图显示驱动草稿预览，且不出现面拾取错误提示。
6. 平移驱动绑定平移副或圆柱副时，驱动预览与已创建图形中的单向箭头都与连接轴线平行且共轴。
7. 标准模式下，选择连接后不会立即创建，点击 `添加` 才创建。
8. 创建后驱动出现在模型树 `驱动` 分类，并在三维视图中保留红色驱动绘制图形。
9. 转动驱动预览与已创建图形中的箭头头部由两条边线组成，不出现单边缺失，且顶部朝上、头部稳定位于右下象限。
10. 选中驱动时，右侧显示 `属性-驱动`，且 `类型` 只读。
11. `驱动类型 / 图标大小 / phi.start / w.start` 修改后可保存。
12. `Alt + 鼠标滚轮` 可调整创建态图标大小。
13. `Esc` 可退出驱动创建态，`Del` 可删除选中驱动。
14. `motion` 导入导出不丢字段，兼容旧字段别名。
